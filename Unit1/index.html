Internet

    Internet is a network of networks
    The Internet is a global system of interconnected computer networks that use the standard internet protocol suit(TCP/IP) to serve billions of users worldwide.
    The Internet is a gloabal data communications system, a vast infrastructure that allows comouters and devices to communicate and share data worldwide.

IP addresses:

    It's a unique 32 bit number assigned to computers connected to the internet.
    IP addresses are written as four 8 bits numbers, separated by periods. eg:- 191.57.126.0

Domain Names

    A domain name typically consists of a second-level domain (SLD) and a top-level domain (TLD) separated by a dot (period)

for example :- google.com
